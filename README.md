# 📊 Customer Churn Intelligence Dashboard

## 🚀 Project Overview
Customer churn is one of the biggest challenges for businesses as it directly impacts revenue and growth.  
This project focuses on analyzing customer data to identify patterns, trends, and key factors contributing to customer churn.

The goal of this dashboard is to transform raw data into meaningful insights that help businesses improve customer retention strategies.

---

## 🎯 Problem Statement
How can businesses identify the key factors that lead to customer churn and take proactive steps to improve retention?

---

## 📌 Dashboard Features
- KPI Cards (Total Customers, Churn Customers, Churn Rate, Retention Rate)
- Churn Analysis by Contract Type
- Churn Analysis by Internet Service
- Impact of Tech Support on Churn
- Interactive Filters (Gender, Senior Citizen, Contract, Internet Service)
- Visual representation using charts and graphs

---

## 💡 Key Insights
- Customers with **month-to-month contracts** show the highest churn rate  
- **Fiber optic users** have higher churn compared to DSL users  
- Customers without **tech support** are more likely to churn  
- Long-term contracts (**1–2 years**) significantly improve retention  
- Overall churn rate is around **26%**, indicating potential business risk
